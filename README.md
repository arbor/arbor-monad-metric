# arbor-monad-metric

[![CircleCI](https://circleci.com/gh/arbor/arbor-monad-metric.svg?style=svg)](https://circleci.com/gh/arbor/arbor-monad-metric)

The `arbor-monad-counter` is retained for backwards compatibility.
