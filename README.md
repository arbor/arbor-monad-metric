# arbor-monad-metric

[![CircleCI](https://circleci.com/gh/arbor/arbor-monad-metric.svg?style=svg)](https://circleci.com/gh/arbor/arbor-monad-metric)

This is a fork of `arbor-monad-counter` into a module namespace to add support for varied metric types.
The `arbor-monad-counter` library is retained for backwards compatibility.

# Release
Bump the version in the `*.cabal` version; create a new commit “New version x.x.x.x`, wait for tagged build in CI and find a link to hackage in the build output and open it.  Log in with hackage credentials if necessary, then click “publish candidate”
